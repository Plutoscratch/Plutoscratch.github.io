                      # Hexo+Aurora创建博客

                      大家好，我是冥王星

                      今天给大家带来Hexo+Aurora创建博客的教程

                      # 前期准备

                      • Git

                      一个控制台，跟cmd差不多，可以用于把博客部署到托管平台上 下载地址：https://www.git-scm.com/downloadsopen in new window

                      • Nodejs

                      Hexo是基于Nodejs开发的，博客用它来生成文件 下载地址：https://nodejs.org/zh-cn/open in new window

                      Custom header

                      注意事项

                      • Nodejs推荐下载长期维护版(LTS)，因为最新版有很多不便

                      # 环境搭建

                      • 首先，分别安装Git和Nodejs，基本每一步的直接点击跳过即可

                      • 安装完成后Win+R输入cmd进入命令控制台，输入以下命令,来检查Nodejs是否安装成功：

                      node -v
npm -v
                      1
                      2

                      接下来，创建一个Blog文件夹，进入这个文件夹，点击右键使用Git Bash Here,然后输入以下命令：

                      npm install -g hexo-cli
hexo init myblog
cd myblog
npm install
//这个myblog文件夹其实是可以改名字的，不过我为了小白考虑，提供这个一键生成Blog文件的命令
                      1
                      2
                      3
                      4
                      5

                      如果安装无误的话，你的Blog(myblog)文件夹的结构应该是这样的：

                      • node_modules: 依赖包

                      • public：存放生成的页面

                      • scaffolds：生成文章的一些模板

                      • source：用来存放你的文章

                      • themes：主题

                      • _config.yml: 博客的配置文件

                      接下来，在Git Bash界面输入以下命令，打开Hexo自带的服务器：

                      hexo g && hexo s
                      1

                      • 在浏览器输入loaclhost:4000就可以看见自己的Blog了

                      接下来，我们就正式进入Aurora主题的安装教程了，请大家检查一下各个步骤

                      # 安装主题

                      1. Win+R打开cmd,输入以下命令安装Aurora主题
                      npm install hexo-theme-aurora --save
                      1

                      Custom header

                      注意事项

                      • 注意：安装完成后请检查是否有_confing.yml文件，没有请自行去官方文档获取安装教程。
                      1. 第二步，打开在 Hexo 根目录下的 _config.yml
                      • 修改 permalink 参数为 /post/:title.html
                      • 把 highlight 的启用改为false
                      • 把 prismjs 的启用改为true
                      • 把 prismjs 下的 preprocess 改为 false
                      permalink: /post/:title.html
highlight: false
prismjs: true
  preprocess: false
#注意：冒号后面需要有一个空格
